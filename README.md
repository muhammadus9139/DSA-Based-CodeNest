##CodeNest

CodeNest is a JavaFX-based mini GitHub clone and AI Copilot-powered development platform built using Java and Data Structures & Algorithms concepts. It simulates core GitHub functionalities along with an integrated AI assistant for code review and suggestions using the Groq API.

Features
User Authentication (Login & Signup)
GitHub-like Dashboard with Trending Feed
Repository Creation, Editing & Deletion
File Upload & File Creation from System
Real-Time Commit Tracking
Activity Tracker (Commit History per day)
Profile Customization with Profile Photos
Popular Repositories & Trending Data (via REST APIs)
AI Copilot for Code Review & Suggestions (Groq API)
JavaFX Desktop Application UI
Multiple Data Structures & Algorithms implementations

Tech Stack
Java
JavaFX
REST APIs
Groq AI API
Data Structures & Algorithms
File Handling
Object-Oriented Programming

Project Highlights

Installation
Clone the repository
Open in IntelliJ IDEA 
Configure JavaFX libraries
Add Vm option
Add required API keys (Gson, Groq, REST APIs)
Run the application
